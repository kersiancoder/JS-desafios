Reglas de negocio

Solo el usuario que registró el vehículo puede modificarlo y/o eliminarlo como también ver el precio de compra original.

Para solicitar un test drive no debe estar ingresado el usuario que registró dicho vehículo.

Para vehículos nuevos (0 Km) el precio de venta se calcula al 200% del precio de compra.

Para vehículos usados el precio de venta se calcula al 150% del precio de compra.

El botón de salir no borrará los vehículos creados previamente, solo deslogueará al usuario y se podrá ingresar con otro usuario para crear más vehículos.

El botón de reiniciar programa borrará todos los vehículos y usuarios del Array y el localStorage, para luego dejar creados únicamente los que estan en el JSON.
